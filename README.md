# Key bindings for create new file and folders
```
[
    {
        "key": "ctrl+n",
        "command": "explorer.newFile",
        "when": "!editorFocus"
    },
    {
        "key": "ctrl+shift+n",
        "command": "explorer.newFolder",
        "when": "!editorFocus"
    },
    {
        "key": "ctrl+d",
        "command": "editor.action.copyLinesDownAction",
        "when": "editorFocus"
    }
]
```
